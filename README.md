Pingu0in — Minimal Generative art Linktree

A fully custom minimal, scroll-based Linktree-style landing page built with pure HTML, CSS, and JavaScript — enhanced with generative art powered by p5.js.
Preview
A vertical snap-scrolling experience where each section represents a platform:
- Instagram
- TikTok
- GitHub
- Twitter
- Twitch
- YouTube
- Discord

Each platform includes a unique generative artwork, animated card reveal, custom-styled button, and large background keyword watermark.
Concept
This project explores:
- Code as visual language
- Noise & randomness as design elements
- Minimal typography contrast
- Cinematic scrolling
- Anti-design inspired layout

Each section includes a custom p5.js sketch reflecting the platform’s identity.
Tech Stack
- HTML5
- CSS3 (Custom Properties + Scroll Snap)
- Vanilla JavaScript
- p5.js (Generative Art)
- Font Awesome (Icons)
- Google Fonts (DM Serif Display, Syne, Space Mono)
Features
Custom Cursor:
- Smooth trailing ring animation
- Expands on hover over links
- Uses mix-blend-mode: difference

Scroll System:
- Scroll snap layout
- Dynamic progress bar
- Section counter (01 / 07)
- Smooth card reveal with IntersectionObserver

Generative Art System:
- Separate p5 instance per section
- Seeded randomness
- Static rendering (noLoop)
- Monochrome palette
